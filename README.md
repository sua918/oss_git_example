### Week1-1 강의 개요 (_강의계획서_)
* 목표
  1. 오픈소스 소프트웨어 개발을 위한 기본 개념과 도구를 중점으로 배움.
  2. 애자일 기반 소프트웨어 개발 방법을 공부함.
  3. 낯선 소프트웨어 개발 환경 및 도구를 스스로 배우는 태도를 배움.
* 수업 방식
  * 플립러닝 방식으로 진행함.
  * 매 강의 시작시 10분 퀴즈 진행함.
* 평가 방식
  * 중간고사 30%
  * 기말고사 40%
  * 개별과제 20%
  * 출석 10%
